# SDLuna
A SDL2 bind for Lua

## Environment 
* C++11

## Dependence
* [SDL2.0](https://www.libsdl.org/)
* [Lua5.3](http://www.lua.org/)
* [Lua Magic](https://github.com/hubenchang0515/Lua-Magic)
